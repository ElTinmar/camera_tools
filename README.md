# Ximea

Install xiAPI

```
wget https://www.ximea.com/downloads/recent/XIMEA_Linux_SP.tgz
tar xzf XIMEA_Linux_SP.tgz
cd package
./install -pcie
cp api/Python/v3/ximea <path/to/python/site-packages/>
```
